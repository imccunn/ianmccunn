Web Audio API 1

Synthesizing Chords of Random non-tempered Frequencies

This small proof of concept allows you to use Web Audio API oscillators to generate a chord of n pitches within a frequency range.

In this case, a 'chord' is simply meant here to be a collection of frequencies played temporally in sync. Select the number of pitches you want to generate, the minimum and maximum possible requencies and the duration. Hit 'Play Chord' to hear the procedurally generated chord. A slight ADSR envelope is applied to each frequency in a random fashion (both for interest and smooth attack and release).
