HTML Shorthand Instrumentation Generator

This tool provides a quick way to generate the HTML code necessary to display shorthand orchestral instrumentation. Select instruments to be included in your listing, number of each, whether or not a section should be delineated by a '/', and what text should be displayed on hover of instruments in listing.

Example:

Instrumentation: (2.2.1.1 / 4.3.2.1 / Pc.+4.Pf. / 16.14.12.12.10)
